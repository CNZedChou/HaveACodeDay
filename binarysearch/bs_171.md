# [Reverse Words](https://binarysearch.com/problems/Reverse-Words)

- Level：{Easy}
- Tags：{String}

## Idea

- [1] Use **stringstream** to split the string into words.

### Implementation - 1

- Complexity：
  - Time O(N)
  - Space O(N)

``` c++
string solve(string sentence) {
    stringstream all(sentence);
    string res = "";
    string word = "";
    while (all >> word) {
        res = word + " " + res;
    }
    return res.substr(0, res.size() - 1);
}

```
