# [最近的请求次数](https://leetcode-cn.com/problems/number-of-recent-calls/)

- Level：{Easy}
- Tags：{Queue}

## Idea

- [1] Use a **queue** to maintain the window of ping. Return the size of the queue.

### Implementation - 1

- Complexity：
  - Time O(N)
  - Space O(N)

``` c++
class RecentCounter {
public:
    queue<int> q;
    RecentCounter() {

    }
    
    int ping(int t) {
        q.push(t);
        while (t - q.front() > 3000) q.pop();
        return q.size();
    }
};
